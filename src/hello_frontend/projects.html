Tic-Tac-Toe-Game

This project is a simple console-based Tic-Tac-Toe game developed using the C programming language. The game allows two players to take turns marking spaces on a 3x3 grid, aiming to get three of their marks in a row, column, or diagonal to win the game..

The program handles user input, checks for win conditions, and ensures valid moves throughout the game. It was built using fundamental C programming concepts like arrays, loops, functions, and conditional statements to manage the game logic.

Time-Table

This project is a simple and responsive time-table designed using HTML and CSS. It visually represents a weekly schedule, making it easy to organize tasks, classes, or events for each day of the week. The layout adapts to different screen sizes, providing an optimal user experience on both desktop and mobile devices.

The time-table is structured using HTML tables for easy organization of rows and columns, while CSS is used to style the table, making it visually appealing and easy to navigate. Additional styling ensures that the table is clean, color-coded, and user-friendly.
